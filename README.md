# Good Morning Web

A simple form to collect name, phone, and email.

## What it does
- Shows "Good Morning" page
- Has a form with 3 fields: name, phone, email
- Sends data to backend when you click submit

## How to run

1. Clone this repo
2. Run `npm install`
3. Run `npm run dev`
4. Open http://localhost:3000

## Tech used
- Next.js
- React
- CSS
